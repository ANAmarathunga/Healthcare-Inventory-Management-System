# 🏥 Healthcare & Inventory Management System (IPS System)

A comprehensive, full-stack enterprise-grade medical ERP solution designed to optimize clinical workflows, manage pharmacy stocks, and deliver real-time data metrics.

## ✨ Core Features
* 📊 **Executive BI Performance Dashboard:** Features advanced Data Mining for Medicine Usage Trends and Predictive Analytics for forecasting Patient Volumes using Recharts.
* 🧭 **Modern Sidebar Layout:** Fully responsive, dark-themed navigation system embedded with secured authentication states.
* 🩺 **Doctor & Medical Staff Directory:** Full CRUD operations with centralized input grids directly syncing with cloud protocols.
* 📋 **Patient Registration Ledger:** Tracks active patient profiles, unique identifications (NIC), and medical metadata securely.
* 💊 **Pharmacy Inventory Stock Tracker:** Live ledger management monitoring available stocks, batch rates, and statutory pharmaceutical expiration boundaries.
* 📅 **Clinical Appointment Allocator:** Real-time matrix mapping for locking scheduling windows across dedicated wards.
* 💳 **Institutional Billing Hub:** Employs dynamic mathematical regressions calculating a baseline subtotal synchronized with a live **10% PL/SQL Tax Valuation** script.

## 🛠️ Tech Stack
* **Frontend:** React, HTML5, CSS3 (Glassmorphic Styles), Bootstrap 5, Lucide Icons, Recharts.
* **Backend:** Node.js, Express.js.
* **Database Architecture:** Oracle Cloud Infrastructure (OCI) hosted relational databases via Oracle REST Data Services (ORDS).
